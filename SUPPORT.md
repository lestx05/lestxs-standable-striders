# Support

Standable Striders is maintained as a small open-source project. Community
support is provided through GitHub Issues; no response-time guarantee or
service-level agreement is offered.

## Supported releases

| Mod line | Minecraft | Support status |
| --- | --- | --- |
| 1.0.x | 26.2 | Current |
| 0.2.x | 1.21.11 | Legacy; fixes are at maintainer discretion |

Use an official release that matches your Minecraft version. Modified builds,
unsupported dependency combinations, and large modpacks may still be
investigated, but reproducing the problem in a minimal profile will usually be
required.

## Where to ask

Open the
[issue chooser](https://github.com/lestx05/lestxs-standable-striders/issues/new/choose)
and select:

- **Bug report** when existing behavior is broken and reproducible.
- **Compatibility report** when the problem appears only with another mod.
- **Feature request** for a proposed behavior or enhancement.
- **Support question** for installation, updating, or usage help.

Before opening a report, search open and closed issues and read the
[README](README.md). Include exact versions for Minecraft, Standable Striders,
Fabric Loader, Fabric API, and Java. Logs and crash reports should be attached
as files rather than pasted as very long text.

## Security and conduct reports

Never place an undisclosed vulnerability, access token, private server address,
or other sensitive information in a public issue. Security reports belong in
the private channel described in [SECURITY.md](SECURITY.md). Conduct reports
belong in the private channel described in
[CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md).
