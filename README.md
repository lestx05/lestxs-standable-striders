# lestx's Standable Striders

Stand on top of vanilla Striders with platform-like behavior inspired by the Happy Ghast.

## About

**lestx's Standable Striders** makes Striders more useful in the Nether by letting them work as a standable surface instead of only a lava mount.

The goal is simple: expand the utility of vanilla Striders without changing their identity as a vanilla mob.

## Features

- Stand on top of Striders
- Platform-like behavior inspired by the Happy Ghast
- Small, focused gameplay change
- Keeps the Strider's vanilla role intact

## Environment

- **Minecraft:** 1.21.11
- **Loader:** Fabric
- **Client-side:** Optional
- **Server-side:** Required

## Installation

1. Install the correct mod loader for your Minecraft version.
2. Download the correct version of this mod.
3. Place the `.jar` file in your `mods` folder.
4. Install any required dependencies listed on the release page.
